Cómo jugar.

  • Para comenzar a jugar al Kingdom Hearts Breakout tendrás que pulsar la tecla espacio para que la pelota empiece a moverse.
  • Para movernos usaremos las flechas de direccion derecha e izquierda.
  • Cuando la pelota toque el suelo, perderemos una vida.
  • Cuando nuestras vidas lleguen a 0, perderemos el juego.
